# Dragino LoRa Shield v1.2 + Arduino Uno

## Bonus Source Code (Arduino IDE)

| # | Folder | Fungsi |
|---|--------|--------|
| 01 | `01-lora-uart` | Kirim teks satu arah — dasar TX/RX, cocok buat pemula |
| 02 | `02-Lora-UART-LED_notif` | TX + RX non-blocking dengan notifikasi LED |
| 03 | `03-Lora-peer-to-peer` | Ping-Pong dua arah antar dua board |
| 04 | `04-lora-ack` | Kirim data + tunggu konfirmasi ACK dari receiver |
| 05 | `05-master-slave-3node` | Master polling 2 Slave bergantian (round-robin) |
